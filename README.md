# 🪙 Crypto Whisperer Bot

A lightweight Telegram bot that fetches the current Bitcoin price in USD using the free CoinGecko API.

## ✨ Features

- Real-time Bitcoin price in USD.
- Simple, fast, and reliable.
- Error handling for API downtimes.

## 🛠️ Tech Stack

- **Python 3.9+**
- **python-telegram-bot**
- **Requests**
- **CoinGecko API**
- **python-dotenv**
